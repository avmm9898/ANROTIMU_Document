

## CH Center

包含所有產品的

- X Y Z 多軸即時數據、波形圖

- 3D 姿態顯示

- CSV 數據紀錄

- 模組參數設定，如採樣率、鮑率、ID、無線節點 GWID


如有發現問題請回報 hipnuctw@gmail.com
如遇上執行 GUI 提示缺少 msvcpXXX.dll 的狀況，請安裝 VC_redist.x86.exe
